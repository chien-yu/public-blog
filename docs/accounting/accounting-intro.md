# 記帳

記帳聽起來是非常瑣碎的事，更高層次的目標應該是財務管理，只是沒有紀錄，就也無從管理起。關於要不要記帳這件事就很主觀了，小時候長輩耳提面命要我們養成習慣，但長大後社會充斥著另一種論調：「記帳會讓你越記越窮！」、「財富是賺出來的，不是省出來的！」，彷彿太計較花費反而限制了格局。

但我認為，記帳不應該被簡化為省錢的工具。對我而言，它更像是一種人生哲學的實踐。就像寫日記是為了保存生活的樣貌，記帳格式將生活的軌跡用另一種方式被保存下來。

### 我該記帳嗎？
如果你常常月光，那應該早就有在記帳了吧，不然感覺你隨時都有可能繳不出帳單。如果你很肯定自己不會月光，看似記帳是多此一舉，反正銀行帳戶怎麼樣都不會花完。

但這正是我最想探討的盲點——你以為的「足夠」，會不會其實是一種「過度保守」？

部分人因為不清楚自己的資產極限，所以生活開銷上會非常小心地不敢花錢。說不定其實你的資產是可以支撐更豐富的人生體驗，但你卻渾然不知。讓我想到爸媽那一代人，省吃儉用一輩子，但年紀大了後，卻不知道怎麼運用他們的資產。
