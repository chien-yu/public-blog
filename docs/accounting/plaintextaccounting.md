# 為什麼你該開始「純文字記帳」？
其實在推坑這個工具之前，應該要先去探討為什麼我們會需要記帳，但這是另一個坑了。這篇文章是來推薦已經有在記帳的人，一個其實很老但對我們卻新穎的方式：純文字記帳。在手機上搜尋就可以看到各種介面可愛，主打清楚明瞭的 App，但我想推薦的是純文字記帳。

## 操作介面差異
手機上處理複雜資料太困難，可以是電腦上都能處理才方便。還得要是能跨平台到不同電腦都可以同步，所以要有一個中心的資料庫是自己方便管理的，跨平台也能方便運行資料處理。因為資料就是一堆文字檔案在一個資料夾裡面，備份到任何雲端系統都很容易，要進行的資料處理只有錯誤偵測，或是想要看報表時的圖像化工具，安裝起來也不算難，之後就可以在瀏覽器直接看了。

## 寫入方式差異
資料都是在純文字檔，可以自己分類到不同的檔案中，分門別類管理。像是一個你隨時可以看見的資料庫，雖然你也可以使用正式的資料庫，但如果是用現有的幾個開源軟體，不用另外去想著要怎麼定義自己的資料庫結構，還可以輕鬆的加上各種格式的註釋。

## 資料保存
都是文字檔案可以簡單的上傳到各種雲端系統，想要直接用 git 版本控制也很方便。

## 方法差異
用商用軟體記帳，最陽春的是讓你手動輸入你每天的花費，應該主要是現金交易。當你給了這些應用程式信用卡帳戶權限後，信用卡交易倒是可以每天自動同步跟分類。但能做的分類有幾項，能怎麼設計分類，分類可以分幾層的自由度不確定。

### 消費之外
當一般人記帳只做到追蹤消費時，你可以輕易地連資產變化和利潤都記錄下來。
