# 分析一首歌

記錄一下現階段研究一首歌的過程。

1. 在 Youtube 找[原唱](https://www.youtube.com/watch?v=2dXwlykNemw&list=RD2dXwlykNemw&start_radio=1)
2. 【找主調】聽歌的時候在吉他上每個音都試，找到Ｃ調的音都和諧。
3. 【找和弦進行】因為現在只會彈 C 調的和弦，所以有[Ｃ調的樂譜](https://www.youtube.com/watch?v=4kbr5hDKW-c)最方便了，這首歌也剛好是Ｃ調。
4. 【找旋律】可以一個音一個音聽，但就更考驗聽力了。
    1. 要檢查有沒找錯時就用 AI 分離音軌，Moises 看起來夠方便。![moises](moises.png)
    2. 把音軌都下載，人聲丟到 Tony，就可以看到每個音的音高。![tony](tony.png)
5. 【找節奏】有時候拍點聽不清楚，想要確認的話可以看波形圖。但在 Moises 不確定怎麼弄，就把鼓跟人聲都匯進 Audicity。![audicity](audicity.png)
