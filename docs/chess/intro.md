# 序章

早已不記得第一次接觸西洋棋是什麼時候了。我有印象的第一次，是在工作第一年的感恩節。當時一位在實習時就認識的同事邀請我去他家聚餐，在客廳角落的一張桌子上就擺著一副西洋棋盤。我和他的哥哥下了一局，當時的我只知道棋子的基本走法，什麼「王車易位」、「吃過路兵」這類特殊規則都不知道，任何一個開局也不懂。單純靠著計算贏了那盤棋，現在回想起來有點不好意思，去別人家作客還贏了熱情的主人。

後來再次接觸西洋棋，是在一次台灣人的聚會中，不知如何發現朋友的孩子也下棋，可能是那次活動無聊吧，我們就下了一盤。後來的聚會他只要有出現，我們就會躲到後面去，在我的手機上下棋。那時候我還是沒看過任何教學影片，畢竟會下棋的時間還是太少了。

因為有了在手機上下棋的經驗，搭飛機沒有網路無聊到極致時，就會把 App 打開找離線模式來打發時間。居然有兩次，鄰座的乘客看到我的螢幕後，主動搭話問能不能一起玩。意外成為社交利器，兩次都下超過半個小時，也順勢聊開了。

真正的轉折來自在另一位住家附近朋友的孩子。他在幼稚園學會了西洋棋，邀請我去跟他下棋。在跟他下棋的時候想到，以後可能有很多機會陪他下棋，不如我學習一點開局方法，也可以來教他幫他進步。就這樣我開始在 YouTube 上看西洋棋的教學影片。到現在已經會不時被推薦西洋棋影片，手機上也裝了西洋棋的 App，現在看這週的手機紀錄，已經一週下 13 個小時了。
